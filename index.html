Git

Список команд git которые мы разобрали на уроке

  1. git add .
  2. git commit -m " поле коментария "
  3. git push (закидывает изменения в твой репозиторий)
  4. git fetch (скачивает все актуальные изменения файла с репозитории)
  5. git remote update (Не до конца понимаю различие с fetch но суть вроде та же )
